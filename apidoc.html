Documentazione API Biblioteca Online

API Create (Inserimento di un nuovo libro)

Endpoint: /nuovo-libro

Richiesta: POST con i seguenti parametri:

  • titolo (stringa): Titolo del libro
  • autore (stringa): Autore del libro
  • genere (stringa): Genere del libro
  • data_di_pubblicazione (data): Data di pubblicazione del libro
  • copie_disponibili (numero intero): Copie disponibili del libro

Esempio di utilizzo:

POST /nuovo-libro
{
    "titolo": "Il Nome della Rosa",
    "autore": "Umberto Eco",
    "genere": "Storico",
    "data_di_pubblicazione": "1980-09-01",
    "copie_disponibili": 5
}

Esempio di risposta:

{
    "messaggio": "Libro inserito con successo",
    "response": 1
}

API Read (Visualizzazione dei libri)

Endpoint: /visualizza-libri

Richiesta: GET senza parametri

Esempio di utilizzo:

GET /visualizza-libri

Esempio di risposta:

[
    {
        "id": 1,
        "titolo": "Il Nome della Rosa",
        "autore": "Umberto Eco",
        "genere": "Storico",
        "data_di_pubblicazione": "1980-09-01",
        "copie_disponibili": 5
    },
    {
        "id": 2,
        "titolo": "1984",
        "autore": "George Orwell",
        "genere": "Distopico",
        "data_di_pubblicazione": "1949-06-08",
        "copie_disponibili": 3
    }
]

Visualizzazione di un libro specifico:

Endpoint: /visualizza-libro/{id}

Esempio di utilizzo:

GET /visualizza-libro/1

Esempio di risposta:

{
    "id": 1,
    "titolo": "Il Nome della Rosa",
    "autore": "Umberto Eco",
    "genere": "Storico",
    "data_di_pubblicazione": "1980-09-01",
    "copie_disponibili": 5
}

API Update (Modifica delle copie disponibili di un libro)

Endpoint: /modifica-libro

Richiesta: PUT con i seguenti parametri:

  • id (numero intero): ID del libro da modificare
  • copie_disponibili (numero intero): Nuovo numero di copie disponibili

Esempio di utilizzo:

PUT /modifica-libro
{
    "id": 1,
    "copie_disponibili": 10
}

Esempio di risposta:

{
    "messaggio": "Libro modificato con successo",
    "response": 1
}

API Delete (Eliminazione di un libro)

Endpoint: /elimina-libro

Richiesta: DELETE con il parametro:

  • id (numero intero): ID del libro da eliminare

Esempio di utilizzo:

DELETE /elimina-libro
{
    "id": 1
}

Esempio di risposta:

{
    "messaggio": "Libro eliminato con successo",
    "response": 1
}
